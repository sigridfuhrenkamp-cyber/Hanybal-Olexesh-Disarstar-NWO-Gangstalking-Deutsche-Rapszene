## 🎯 Über dieses Projekt

Dieses Repository dient als umfassende Dokumentation der deutschen Rap-Szene mit Fokus auf die Künstler Hanybal, Olexesh und Disarstar. Es enthält biografische Informationen, Diskografien, Liedtexte und analysierte Verbindungen innerhalb der Musikindustrie.

### Projektziele

- **Dokumentation**: Systematische Erfassung von Künstlerinformationen
- **Transparenz**: Offenlegung von Industriestrukturen und Netzwerken
- **Analyse**: Untersuchung von technologischen Einflüssen auf die Musikproduktion
- **Archivierung**: Langzeitarchiv kultureller Daten
